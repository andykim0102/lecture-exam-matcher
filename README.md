🩺 Med-Study OS Pro

의대생을 위한 AI 학습 파트너 (Updated Version)

Med-Study OS는 방대한 강의록과 족보(기출문제) 사이를 연결해주는 스마트 학습 도구입니다. 강의록을 공부할 때, 해당 페이지와 관련된 기출문제를 AI가 자동으로 찾아주고 해설해줍니다.

✨ 새로워진 기능 (v2.0)

💾 데이터 영구 저장: 이제 브라우저를 닫아도 업로드한 족보 데이터가 사라지지 않습니다 (jokbo_db.json에 저장).

🎨 직관적인 UI: 대시보드에서 학습 현황을 한눈에 보고, 과목별로 체계적으로 관리할 수 있습니다.

🧠 문맥 인식 AI: 단순히 문제를 푸는 것을 넘어, 현재 보고 있는 강의록의 내용과 연관 지어 해설합니다.

👁️ OCR 통합: 스캔된 PDF(이미지)도 AI가 글자를 인식하여 분석합니다.

🛠️ 설치 및 실행

필수 라이브러리 설치

pip install -r requirements.txt


앱 실행

streamlit run app.py
